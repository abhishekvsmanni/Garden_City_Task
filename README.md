# Garden_City_Task
UA Profit Forecasting Project
A data science task to forecast 56-day cohort profit using only the first 4 days of marketing data.

Project Highlights:

Exploratory Data Analysis: Visualized spend vs. installs and revenue maturity.

Predictive Modeling: Built a Linear Regression model with a Mean Absolute Deviation (MAD) evaluation metric.

Business Insight: Detailed how managers can use 4-day forecasts to scale winning campaigns and cut underperforming ones.
